# GDPR Enforcement Analysis (2018–2022)

## Project Overview
This project analyzes GDPR enforcement data to identify compliance risks across different countries and types of violations.

## Key Findings
- GDPR fines are highly concentrated in a small number of countries, especially Ireland and Luxembourg.
- Violations of fundamental principles (Art.5) result in the highest financial penalties.
- Frequent violations (e.g., Art.6) are not necessarily the most severe.
- Risk analysis shows a clear distinction between high-frequency and high-impact violations.

## Methodology
- Data cleaning using pandas
- Aggregation (groupby, value_counts)
- Risk matrix visualization
- Time trend analysis

## Legal Insight
The analysis highlights the importance of:
- GDPR fundamental principles (Art.5)
- Transparency obligations (Art.13/14)
- Security measures (Art.32)

It also reflects the impact of the "one-stop-shop" mechanism on enforcement distribution.

## 🛠 Tools

Data sourced from Kaggle GDPR fines dataset (2018–2022)
